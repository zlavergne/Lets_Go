Strings and Characters

byte

  • Alias for uint8

rune

  • Alias for int32
  • Also represents a Unicode point

string

  • Immutable
  • Arbitrary slice of bytes (Not necessarily UTF-8 or ASCII)
  • Contains a length and a pointer to bytes

Arrays and Slices

Arrays

  • Fixed number of contiguous elements of a single type
  • Written as: [20]int

Slices

  • Describe an underlying array
  • Do not require a specific number of elements
  • Written without an integer size: []int
[5]

Functions

Parameters are always passed by value, so use pointers if you need to modify values.

  • We can name return values and define multiple return values:
func divide(a, b int) (quotient, remainder int) {
    quotient = a / b
    remainder = a % b
    return
}
  • Anonymous functions and closures are possible:
var fn = func(a int) int {
    return a
}
  • Variadic functions are possible
func f(b ...int) (length int) {
    length = len(b)
    return
}
[6]

Structs

  • Member visibility follows
  • Anonymous structs are allowed:
var a = struct{
    length int
    width int
    height int
}{5,5, 5}
  • Methods are defined outside of the struct:
func (c cube) volume() int {
    return c.length * c.width * c.height
}
[7]

Interfaces and Channels

Interfaces

  • Specifies a set of methods that a struct must define
  • interface{} matches all types since all types implement 0 or more methods. Equivalent to void* in C.

Channels

  • Mechanism for communication between functions executing concurrently
  • Defines a direction of information and the type of the messages being sent
chan   int  // Integers can be sent or received
chan<- int  // Integers can only be sent
<-chan int  // Integers can only be received
